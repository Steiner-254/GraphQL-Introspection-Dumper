# GraphQL-Introspection-Dumper-SCript
GraphQL Introspection Dumper In Bash

## Overview
This bash script performs a GraphQL introspection query and saves the result to a JSON file.

## Usage
1. Set the GraphQL endpoint in the script.
2. Save the file as ```introspection.sh```
3. Grant the script execution permisions ``` $ chmod +x instrospection.sh ```
4. Run the script to perform the introspection query.

```bash
./introspection.sh
